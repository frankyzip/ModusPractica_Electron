# Icon Placeholder

Place your application icons here:

- `icon.ico` - Windows (256x256 or multiple sizes)
- `icon.icns` - macOS
- `icon.png` - Linux (512x512 PNG with transparency)

You can create a simple icon from text or use a music-related symbol.

Free icon resources:

- https://www.flaticon.com/
- https://icons8.com/
- https://www.iconfinder.com/

Icon generators:

- https://www.electron.build/icons (recommended for Electron)
- https://iconverticons.com/online/
